# Node.js Projects

This directory contains Node.js and JavaScript ecosystem projects organized by category.

## Structure

```
nodejs/
├── fundamentals/     # Basic Node.js concepts
├── typescript/       # TypeScript projects
├── graphql/          # GraphQL implementations
├── react/            # React projects
├── nextjs/           # Next.js projects
├── orm/              # ORM examples
└── infrastructure/   # DevOps and infrastructure
    ├── docker/       # Docker configurations
    ├── kafka/        # Kafka implementations
    ├── redis/        # Redis cache
    └── upload-imagem/ # Image upload system
```

### GraphQL

- **graphql-fundamentos**: GraphQL fundamentals (Decode #019)
- **nodejs-graphql-aplicado**: GraphQL applied (Diego Fernandes)

### React

- **react-hooks-typescript**: React Hooks with TypeScript (Masterclass #04)
- **React Router Study**: React Router studies

### Next.js

- **class-nextjs**: Next.js classes and examples

### ORM

- **nodejs-crud-typeorm**: CRUD with TypeORM, Express, PostgreSQL

### Infrastructure

- **docker**: Node.js Docker environment
- **kafka**: Kafka POC with producer/consumer
- **redis**: Redis cache implementation
- **upload-imagem**: Image upload system
