## 🏗️ CQRS-Events-in-Node.js

### Projeto Principal

1. **CQRS Events Lab**
   - Descrição: Simulação de carrinho de compras usando CQRS
   - Baseado em: [CQRS: What? Why? How?](https://sderosiaux.medium.com/cqrs-what-why-how-945543482313)
   - Tecnologias:
     - Node.js (Write System, Read System)
     - Python/Django (Read System alternativo)
     - PostgreSQL (múltiplos bancos)
     - Docker Compose
     - gRPC Adapter
     - Event Sourcing
   - Estrutura:
     - `src/` - Versão principal
     - `src2.0/` - Versão 2.0
     - `src3.0/` - Versão 3.0
     - `grpc-adapter/` - Adaptador gRPC
     - `mocks/` - Mocks para testes
   - Status: Projeto completo e complexo
