# AI/ML Projects

This directory contains Artificial Intelligence and Machine Learning projects, experiments, and course materials.

## Structure

```
ai-ml/
├── tools/              # AI tool integrations and experiments
│   ├── chatbots/      # Chatbot implementations
│   ├── dashboards/    # AI-powered dashboards
│   ├── ocr/           # Optical Character Recognition
│   ├── langchain/     # LangChain experiments
│   ├── audio/         # Audio generation and processing
│   └── ml-basics/     # Basic ML projects
├── algorithms/        # Algorithm implementations
│   └── genetic-algorithms/
├── formations/        # Course materials
│   ├── fiap/          # FIAP course materials
│   └── alura/         # Alura course materials
└── projects/          # Mature projects
```

## Projects

### Tools

- **Chatbots**: OpenAI chatbot implementations
- **Dashboards**: ChatGPT and finance dashboards
- **OCR**: Optical Character Recognition projects
- **LangChain**: LangChain testing and experiments
- **Audio**: Audio generation using AI
- **ML Basics**: Simple machine learning projects

### Algorithms

- **Genetic Algorithms**: Genetic algorithm implementations and studies

### Formations

- **FIAP**: POS FIAP IA para Devs course materials
- **Alura**: Alura AI/ML formation materials

### Projects

- **Imobiliaria-Casas**: Real estate price prediction
- **NoMoreSpams**: Spam detection system
- **Telegram-Robot**: Telegram bot implementation

## Notes

Projects in this directory are learning experiments. When they mature, they may be moved to `ia-ml-knowledge-base`.
