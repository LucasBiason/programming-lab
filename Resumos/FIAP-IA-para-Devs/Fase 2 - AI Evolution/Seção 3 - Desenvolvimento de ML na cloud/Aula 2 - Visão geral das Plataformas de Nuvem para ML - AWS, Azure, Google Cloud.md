# Aula 2 - Visão geral das Plataformas de Nuvem para ML - AWS, Azure, Google Cloud

## Resumo executivo

Esta aula apresenta **visão geral das plataformas de nuvem para ML**: **AWS** (SageMaker: treino, deploy, pipelines; Rekognition, Comprehend, Translate; Lambda e containers), **Azure** (Azure ML: workspace, datasets, compute, deploy; Cognitive Services; Databricks), **Google Cloud** (Vertex AI: treino e deploy unificado; AutoML; BigQuery ML; TPUs). Comparação de conceitos: notebooks gerenciados, experiment tracking, registro de modelos, endpoints em tempo real e batch, integração com dados (S3, Blob, GCS). Escolha conforme ecossistema, custo e tipos de modelo (custom vs AutoML).

**Objetivos de aprendizagem:** Reconhecer ofertas de ML na AWS, Azure e GCP; comparar SageMaker, Azure ML e Vertex AI; citar serviços de IA pronta (Rekognition, Cognitive Services, etc.).

---

## Conceitos-chave (flashcards)

1. **O que é AWS SageMaker?** **R:** Serviço gerenciado para construir, treinar e implantar modelos de ML; notebooks, treino distribuído, deploy em endpoints e batch.
2. **O que é Azure ML?** **R:** Workspace para experimentos, datasets, compute (CPU/GPU), pipelines e deploy; integração com Cognitive Services e Databricks.
3. **O que é Vertex AI (GCP)?** **R:** Plataforma unificada para treino e deploy de modelos; AutoML, treino custom, pipelines, endpoints; suporte a TPUs.

---

## Perguntas para teste de reforço

1. O que é AWS SageMaker? **R:** Serviço gerenciado para construir, treinar e implantar modelos de ML.
2. Cite um serviço de ML da Azure. **R:** Azure ML (workspace, treino, deploy), Cognitive Services, Databricks.
3. O que oferece o Vertex AI na GCP? **R:** Treino e deploy unificado, AutoML, TPUs, pipelines.
4. Que fatores considerar ao escolher uma plataforma de ML na nuvem? **R:** Ecossistema, custo, tipo de modelo (custom vs AutoML).
