# Aula 01 - Quick Wins

**Módulo:** Fundamentos IA Generativa | **Curso:** Full Cycle MBA

---

## Resumo executivo

- **Objetivo:** Introdução a Quick Wins com IA — ferramentas e conceitos para ganhar produtividade imediata.
- **Ferramentas de desenvolvimento com IA:**
  - **IDEs:** Cursor, Windsurf, VSCode + GitHub Copilot, JetBrains
  - **Agentes CLI:** Claude Code, Gemini, Codex
- **Ideia central:** Documentos de contexto e regras tornam a IA muito mais eficaz; cada novo chat é como um novo dev entrando no projeto — precisa do contexto certo.

---

## Conceitos-chave (flashcards)

- **P: O que são Quick Wins com IA?**  
  R: Ganhos rápidos de produtividade usando ferramentas de IA (IDEs, agentes CLI) com boa engenharia de contexto.

- **P: Por que escolher uma IDE com IA?**  
  R: Integração direta com código, sugestões em tempo real, autocomplete inteligente, debugging assistido.

- **P: Qual a diferença entre IDE e agente CLI?**  
  R: IDE: edição integrada. Agente CLI: executa tarefas em terminal (Claude Code, Gemini) com acesso a arquivos e ferramentas.

## Perguntas de reforço

1. Basta instalar o Copilot para ter Quick Win? Não; é preciso configurar contexto, regras e fluxo de trabalho.
2. Claude Code e Cursor fazem a mesma coisa? Não; Claude Code é agente em terminal; Cursor é IDE com IA integrada.
3. O que deve estar pronto antes de usar IA no projeto? README, stack, guidelines e documentação de contexto.
