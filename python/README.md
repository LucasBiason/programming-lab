# Python Projects

This directory contains Python projects organized by framework, pattern, testing methodology, and tools.

## Structure

```
python/
├── frameworks/        # Web frameworks
│   ├── django/       # Django projects
│   ├── flask/        # Flask projects
│   └── streamlit/    # Streamlit dashboards
├── patterns/         # Design patterns
│   └── design-patterns/
├── testing/          # Testing methodologies
│   ├── tdd/          # Test-Driven Development
│   └── test-doubles/ # Test doubles and mocks
├── tools/            # Development tools
│   └── environments/ # Virtual environments setup
└── api-integrations/ # API integrations
```

## Projects

### Frameworks

#### Django
- **Environments**: Virtual environment setup with Docker
- **CIDashPyPY**: CI/CD dashboard (4Linux course)
- **RecipeAppApi**: Django REST API with TDD
- **django-and-react-jwt-authentication**: JWT authentication

#### Flask
- **jogoteca**: Game library (Flask parts 1 & 2)
- **FlaskSeries**: E-commerce application (YouTube series)
- **RestFlask-Hotels**: REST API for hotels (Udemy)

#### Streamlit
- **streamlit-quickstart**: Simple dashboard quickstart
- **streamlit-dashboard**: Sales dashboard
- **streamlit-postgres**: PostgreSQL integration
- **streamlit-multipage**: Login and multi-page system

### Patterns

- **design-patterns-python-1**: Design Patterns Python I (Alura)
- **design-patterns-python-2**: Design Patterns Python II (Alura)

### Testing

- **tdd**: Test-Driven Development with Python (Alura)
- **test-doubles**: Test doubles and mocks (Alura)

### Tools

- **environments**: Python virtual environments (pipenv, pyenv, virtualenv, Docker, ASDF)

### API Integrations

- **APIBancoCentral**: Banco Central API integration

## Notes

Projects are organized by course/tutorial source. Each project maintains its original structure and documentation.
